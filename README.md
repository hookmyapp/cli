# @gethookmyapp/cli

HookMyApp CLI — connect WhatsApp Business API in minutes.

## Install

```bash
npm install -g @gethookmyapp/cli
```

Requires Node.js >= 18.

## Quick Start

```bash
hookmyapp --help
hookmyapp --version
```

Run the interactive setup to connect your WhatsApp Business account and start receiving webhooks:

```bash
hookmyapp init
```
